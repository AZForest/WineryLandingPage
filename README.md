# WineryLandingPage

A landing page created for a fictional Winery using HTML and CSS. The page contains no functionality, and was created solely for design purposes. The page includes sections inluding a title section, menu, visit section, tasting section, an additional information section, a wine club section, and a history/philospohy section.
